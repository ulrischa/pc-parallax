pc-parallax

A reusable CSS-only parallax component using native CSS Scroll-driven Animations.

The reusable component CSS is scoped with @layer, @scope, prefixed classes, and typed custom properties.

Multiple layers

CSS-only, scoped with a prefix, @layer, @scope, and local variables.

Reusable by design

For production use, keep the component CSS and replace the demo styles with your project styles.

Spacing, shadows, border radius, images, colors, and text styles are intentionally outside the component core.
